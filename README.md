# Wisperia

Wisperia is a community-driven platform where people can capture, preserve, and share the life lessons that shape who they become. Whether it's a lesson learned from failure, a breakthrough moment, a career experience, or personal growth, Wisperia provides a meaningful space to document those insights and learn from the experiences of others.

In a world where valuable lessons are often forgotten over time, Wisperia helps users turn experiences into lasting wisdom while building a supportive learning community.

## Live Website

**Live Site:** https://wisperia.vercel.app

**Client Application:** https://wisperia-client.vercel.app

---

## Why Wisperia?

Every person carries unique experiences and lessons that can inspire, guide, or help others avoid similar mistakes. Wisperia was built with a simple idea:

> "The lessons we learn today can become guidance for someone tomorrow."

The platform encourages reflection, personal growth, and knowledge sharing by allowing users to create and discover meaningful life lessons from people around the world.

## Running the Project Locally

### Clone the Repository

```bash
git clone <client-repository-url>
git clone <server-repository-url>
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

### Start the Backend Server

```bash
npm start
```

## Design Approach

Wisperia was designed with a focus on simplicity, readability, and meaningful user experiences. The interface aims to create a calm and engaging environment where users can comfortably share personal insights and explore lessons from others.

* Responsive layouts
* User-friendly navigation
